⚡ FILOSOFIA DO CÓDIGO

Este site foi construído com uma filosofia de simplicidade, transparência e "anti-encantamento". A estética minimalista, inspirada na web dos anos 90, é uma decisão de design deliberada — não uma limitação técnica. O objetivo é focar no conteúdo e na estrutura, removendo o ruído visual.

Sem frameworks. Sem build. Sem mistério. Apenas HTML, CSS e JavaScript puros, servidos como arquivos estáticos.
