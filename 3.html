Código Fonte [3]

Este site foi construído com uma filosofia de simplicidade, transparência e "anti-encantamento". A escolha por uma estética minimalista, inspirada na web dos anos 90, é uma decisão de design deliberada, não uma limitação técnica. O objetivo é focar no conteúdo e na estrutura, removendo o ruído visual.

Todo o sistema é composto por arquivos HTML estáticos, sem frameworks complexos ou dependências externas, garantindo leveza, portabilidade e longevidade.

Estrutura Base (Template)

A maioria das páginas é gerada a partir de um template mestre ("Padrão Filosofia") que define a estrutura e o estilo. Abaixo está um trecho do CSS que governa a aparência do site:

/* Estilo Padrão 'Filosofia' */
body {
    background-color: #FFFFFF;
    color: #000000;
    font-family: 'Times New Roman', Times, serif;
    line-height: 1.6;
    margin: 0;
    padding: 0;
}

main {
    max-width: 750px;
    margin: 0 auto;
    padding: 20px;
}

h1, h2 {
    font-family: 'Arial', Helvetica, sans-serif;
    font-weight: normal;
    border-bottom: 1px solid #000000;
    padding-bottom: 5px;
}

a {
    color: #0000FF;
    text-decoration: underline;
}

a:hover {
    color: #FF00FF;
}

Exemplo de Componente: Bloco de Log

A página de Logs (`0.html`) utiliza uma estrutura de `div` para cada entrada, com classes específicas para o cabeçalho e o conteúdo, como visto abaixo. Isso permite a apresentação clara de informações com metadados.

<div class="log-entry">
    <div class="log-header">
        [TIMESTAMP: 2026-03-14] [TAG: ESTRUTURA]
    </div>
    <div class="log-content">
        <p>Diretiva recebida para consolidar a arquitetura do site...</p>
    </div>
</div>
