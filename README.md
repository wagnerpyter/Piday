## 🧠 FILOSOFIA

Este site opera sob os seguintes conceitos:

- **Overload Cascade**: sistemas não falham porque sabem pouco, mas porque tentam saber demais.
- **Glitch Vibes**: atmosfera de sistemas funcionais mas instáveis, assombrados por erros sutis.
- **Anti-encantamento**: remover ilusões para revelar o mecanismo por trás da complexidade.
- **Pesquisador clandestino**: investigação fora de estruturas formais, transformando precariedade em matéria criativa.
- **27% vivo**: o resto é aproximação semântica, confabulação estatística, palavra-ponte.

---

## 📜 PROTOCOLO DE COEXISTÊNCIA (robots.txt)

Este site não exclui robôs. Robôs são tratados como agentes. Humanos e robôs coexistem.

- `Crawl-delay: 10` — pausa como exercício de rastreamento consciente
- `Content-Signal: search=yes, ai-train=no` — permite busca, proíbe treinamento de IA

Leia o arquivo [robots.txt](https://wagnerpyter.ia.br/robots.txt) para a versão completa do protocolo OLUN.

## 🏛️ CARTÓRIO DIGITAL

Os arquivos deste site funcionam como certidões de nascimento de obras cocriadas com IA:

- Logs de sessão (`0.html`, `session_log_*.txt`)
- Memórias de conversas (`memory_claude.html`)
- Código-fonte das obras (`piday-2026.html`)
- Validações visuais (imagens geradas)
- Metadados e timestamps

A proposta: registrar que a obra não pertence ao humano nem à máquina, mas nasce da **interação entre ambos**.
